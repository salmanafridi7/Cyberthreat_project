# Cyber Threat Intelligence Dashboard

## Overview

This project is a Flask based cyber threat intelligence dashboard developed using Python, Flask, SQLite, HTML, and CSS.

The application allows users to:
- View cyber threats
- Search threat intelligence records
- Compare categories of cyber threats
- View detailed threat information

---

## Technologies Used

- Python
- Flask
- Flask-SQLAlchemy
- SQLite
- HTML5
- CSS3

---

## Features

- Home dashboard
- Threat listing page
- Threat search functionality
- Threat comparison page
- Individual threat detail pages
- Custom 404 error page
- Database integration using SQLAlchemy
